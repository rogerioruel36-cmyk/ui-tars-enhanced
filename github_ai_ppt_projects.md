# GitHub AI生成PPT/PDF项目调研报告

基于实际浏览和分析，以下是8个优质的AI生成PPT或PDF的GitHub项目：

## 1. presentation-ai
**链接：** https://github.com/allweonedev/presentation-ai

**技术栈：** TypeScript, Next.js, React, Tailwind CSS, PostgreSQL, Prisma ORM, OpenAI API, Together AI, NextAuth.js, Radix UI, Plate Editor

**功能支持：**
- ✅ PPT支持：是（生成可编辑的演示文稿）
- ✅ PDF支持：否（主要输出为PPT）
- ✅ 图表支持：是（支持图表生成）
- ✅ 背景图支持：是（9个内置主题，支持自定义主题和背景）

**项目描述：** ALLWEONE开源AI演示文稿生成器，是Gamma.app的替代品。支持AI驱动的内容生成、可自定义的幻灯片、实时生成、自动保存、多个主题、自定义主题创建、图像生成、演示模式等功能。

**Stars：** 2567
