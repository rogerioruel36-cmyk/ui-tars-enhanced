# MuleRun 研究报告

## 什么是 MuleRun

MuleRun 是一个 AI Agent（人工智能代理）市场平台，被称为"世界最大的人工智能代理市场"（The World Largest AI Agent Marketplace）。它是一个超级代理（superagent）平台，用户可以通过自然语言聊天的方式，让 AI 完成各种复杂的工作任务。

官方网站：https://mulerun.com

## 核心功能

### 1. AI Agent 市场
- 平台提供 **180+** 个预构建的 AI Agent
- 累计已完成 **120万+** 次 Agent 运行
- 已有 **700+** 位创作者入驻
- Agent 涵盖多个领域：内容创作（Content Creation）、教育（Education）、营销（Marketing）、设计（Design）、生活助手（Life Assistant）、娱乐（Entertainment）等

### 2. Agent Builder（Agent 构建器）
用户可以创建自定义的 AI Agent，主要功能包括：
- **Agent 详情设置**：设置 Agent 名称和指令（Agent Instructions）
- **技能选择（Skills）**：从预构建的技能市场选择需要的技能，或上传自定义技能包（Skill ZIP）
- **高级设置**：
  - MCP（Model Context Protocol）配置：连接外部工具和数据源
  - Assets：上传静态资源（参考图片、文档等）
  - Env：添加环境变量（如 API 密钥）

### 3. API 接口
MuleRun 提供 API 供开发者使用，支持：
- **LLM（大型语言模型）**：OpenAI 格式、Anthropic 格式
- **图像生成**：Midjourney、Nano Banana、Nano Banana Pro、Nano Banana 2、Wan 2 等
- **视频生成**：Kling、Midjourney、Sora 2、Veo 3、Wan 2、Wan 2 Spark 等
- **MCP 服务**：Firecrawl MCP、Context7 MCP

### 4. Creator Program（创作者计划）
- 用户可以注册成为 Creator
- 可以在平台上创建和销售自己的 AI Agent
- 提供 Creator Studio 管理工具
- 支持 n8n Agent 和 Iframe Agent 规格
