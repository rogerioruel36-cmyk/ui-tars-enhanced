# GitHub 自动生成脑图/流程图开源项目

## 项目列表

### 1. Mermaid
**链接**: https://github.com/mermaid-js/mermaid  
**自然语言支持**: ✓ 是  
**描述**: 从文本生成流程图、序列图等图表，类似Markdown的方式  
**Stars**: 86,373  
**主要语言**: TypeScript  

Mermaid是最受欢迎的开源图表生成库，支持通过文本描述语言生成多种类型的图表，包括流程图、序列图、甘特图、类图等。它的语法简洁直观，易于学习和使用。

## 总结

在这8个项目中，**4个项目支持自然语言生成**：

1. **Mermaid** - 通过文本语法生成图表，是最成熟和最受欢迎的选择
2. **Kroki** - 提供统一的API接口，支持多种图表格式
3. **AI-Drawio** - 专门的AI驱动工具，支持从自然语言生成draw.io图表
4. **GenAI Mind Map Flow Builder** - 利用LLM从多种数据源生成脑图

其他4个项目虽然不直接支持自然语言生成，但提供了强大的图表绘制和编辑功能，可以与AI工具集成使用。
